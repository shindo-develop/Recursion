# Productクラス
## very easy
```
あなたは開発チームに所属しており、請求書を管理する仕事を任されています。以下に従って、Product クラスを作成し、テストケースを出力してください。

String title	製品名
double price	製品の価格（ドル）
```
```
テストケース

product1 = new Product ("shampoo", 10)
product1.title --> shampoo
product1.price --> 10

product2 = new Product ("conditioner", 5)
product2.title --> conditioner
product2.price --> 5
```
